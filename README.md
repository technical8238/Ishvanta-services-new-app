# Ishavanta One Call Service

A premium enterprise IT services website built with Angular 21, modern glassmorphism design, responsive page layouts, performance-first best practices, and SEO-ready metadata.

## Local development

Start the development server:

```bash
npm install
npm start
```

Open the browser at `http://localhost:4200/`. The application will automatically reload whenever files change.

## Build for production

```bash
npm run build
```

The optimized production output is generated in the `dist/` folder.

## Production readiness

- Responsive modern UI with glassmorphism and premium spacing
- SEO metadata, Open Graph tags, Twitter card tags, and structured data
- Contact form validation, spam protection, and accessibility
- Robots.txt and sitemap.xml for search engines
- Enterprise UI sections for services, portfolio, process, and testimonials

## Deployment

Deploy the contents of the `dist/` folder to any static hosting provider or Angular SSR setup.

## Backend API

The `backend/` folder contains a .NET API foundation for future contact inquiries and enterprise integrations.
